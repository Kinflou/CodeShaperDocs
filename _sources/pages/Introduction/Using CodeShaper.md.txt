# Using CodeShaper

![CodeShaper](home.png)

Upon opening, in the main menu you have the option to pick recent projects
or any already existing configuration or project

Recent projects show when you open a configuration or a shaping project

Configurations are loaded from AppData/CodeShaper in Windows and
/usr/home/user/ in Linux


## Configurations

These files tell where the shaping project is located and where 
the target files to be shaped are located, they are written in [Human JSON](https://hjson.github.io/) format
